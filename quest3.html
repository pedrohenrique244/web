html - hypertext markup language ou linguagem de marcaçao de hipertexto


O html e uma linguagem de marcaçao utilizada para desenvolvimento de sites.Esta liguagem surgiu junto com o HTTP, ambod possiblitaram a inetrnet ser popularidades

Historia

O html foi criado 1991,por tim berners lee,no CERN((european council for nuclear research) na suiça,inicialmente o html foi projetado para interligar instituiçoes de pesquisa proximos,e compartilhar
documentos com facilidade em 1992,foi liberado a biblioteca de desenvolvimento www(word wide web),uma rede de alcance mundial,que junto com o html proporcionou o uso em escala mundial do
WEB.


Como funciona

OHTML é uma linguagem de marcação. Estas linguagens são constituídas de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria. O HTML tem códigos para criar paginas na web. Estes
códigos que definem o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site. No início era muito complicado aprender HTML, pois eram muitos comandos para fazer algo simples. A
cada nova versão, o HTML fica mais fácil de utilizar, e adquire mais funções. Atualmente qualquer pessoa pode acessar a internet a aprender a construir um site básico em questão de horas, seguindo os passos
de tutoriais e aprendendo as funções de cada código.

O HTML a primeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas á criação de paginas da web, porém o HTML ainda prevalece. Atualmente já é possível
integrar varias linguagens na mesma pagina da Web, sendo possível usar duas ou mais linguagens no mesmo site.

Para criar e editar códigos em HTMLé necessário qualquer editor de texto comum, como bloco de notas. Para testar os códigos, basta salvar o arquivo em formato .HTML e executar. Para o teste é necessário
ter um navegador configurado como padrão. Não é necessária internet, pois o arquivo com os códigos esta na maquina onde esta sendo executado.
